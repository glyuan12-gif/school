## 一、项目概述

### 1.1 产品定位（核心调整 ⚠️）

**一句话定位**：
> 面向国内非技术人群的 **AI知识科普与工具推荐站**，通过优质内容建立信任，引导用户注册使用灵境AI平台，实现推广返佣变现。

**商业本质**：
```
内容引流（免费价值） → 建立信任（专业形象） → 引导转化（灵境AI注册） → 持续返佣（用户消费）
```

### 1.2 目标用户（重新聚焦）

| 用户群体 | 特征 | 核心痛点 | 我们怎么帮他 |
|---------|------|---------|------------|
| **非技术学生** | 不会写代码，想用AI辅助学习 | 不知道怎么用AI、觉得门槛高 | 教程+工具推荐，降低使用门槛 |
| **副业人群** | 想用AI赚钱但无从下手 | 信息过载，不知道哪些靠谱 | 筛选靠谱工具+赚钱案例 |
| **小微企业主** | 想用AI降本增效 | 预算有限，不会技术 | 低成本方案+配置教程 |

### 1.3 核心价值主张（调整后）
- 🤖 **AI零门槛**：手把手教非技术人群使用AI工具
- 💰 **AI能赚钱**：展示AI赚钱案例和实操方法
- 🔗 **一站式入口**：从学习到使用，直接通过我们的平台接入灵境AI
- 📰 **全球视野**：双语对照获取全球AI动态

### 1.4 推广目标
- **目标平台**：[灵境AI](https://www.openaiapp.cc) — 低价AI算力API聚合平台
- **推广链接**：`https://www.openaiapp.cc/register?aff=5FUk`
- **收益模式**：用户通过推广链接注册并消费，获得返佣提成

## 二、功能模块架构（重新设计）

### 设计原则
> **所有内容模块都服务于一个目标：让用户觉得"我需要用AI" → "灵境AI是最方便的入口"**

### 模块一：🤖 AI工具推荐（核心引流模块）

**设计思路**：展示各种AI工具的强大功能，激发用户"我也想用"的欲望，然后引导到灵境AI。

| 功能 | 描述 | 优先级 | MVP |
|------|------|--------|-----|
| 工具分类展示 | 按场景分类（写作/绘图/办公/赚钱等） | P0 | ✅ |
| 工具详情+教程 | 每个工具的详细介绍和使用教程 | P0 | ✅ |
| "立即体验"按钮 | **直接跳转灵境AI注册页（带aff参数）** | P0 | ✅ |
| 使用场景展示 | 真实案例展示AI工具能做什么 | P0 | ✅ |
| 价格对比 | 灵境AI vs 官方价格对比（突出低价优势） | P0 | ✅ |
| 搜索/筛选 | 按关键词、类别、免费/付费筛选 | P1 | ✅ |

**关键设计 — 引流路径**：
```
用户浏览AI工具 → 看到强大功能 → 想要尝试 → 点击"立即体验"
                                              ↓
                                    跳转灵境AI注册页（带aff=5FUk）
                                              ↓
                                        用户注册充值 → 你获得提成
```

**工具分类（按用户场景，非技术分类）**：
- ✍️ **AI写作助手**：写论文、写文案、写邮件、写报告
- 🎨 **AI绘图创作**：做封面、做海报、做Logo、做头像
- 💼 **AI办公提效**：做PPT、做表格、做总结、翻译
- 🎬 **AI视频制作**：做短视频、做配音、做字幕
- 💰 **AI赚钱实战**：AI写小说、AI做自媒体、AI接单
- 📚 **AI学习辅助**：AI辅导、AI翻译、AI笔记
